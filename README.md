# 업도움 CRM

CS 외주 영업을 위한 인하우스 CRM. 채용 사이트에서 CS 채용 공고를 수집하고, 해당 업체에 제안서와 함께 콜드메일을 발송하는 시스템입니다.

## 핵심 기능
- **채용 공고 수집**: URL 수동 등록(잡코리아·사람인·로켓펀치 어댑터) + 워크넷 공식 API
- **업체(리드) CRM**: 미접촉 / 발송 / 회신 / 미팅 / 계약 상태 추적
- **제안서 템플릿**: 변수 치환(`{{company_name}}`, `{{position}}` 등)
- **콜드메일 자동 발송**: Resend 연동 + **옵트아웃 링크 자동 삽입** + **(광고) 표기**
- **발송 안전장치**: 일일 발송 상한, 동일 업체 쿨다운, 수신거부 이력 자동 차단

## ⚠️ 법적 준수 사항
이 시스템은 **정보통신망법 제50조**(영리목적 광고성 정보 전송 제한)를 준수하도록 설계되었습니다:
- 모든 메일 제목에 `(광고)` 자동 표기
- 본문 하단에 발신자 정보·무료 수신거부 링크 자동 삽입
- 수신거부 이력 영구 보관, 재발송 자동 차단
- 발송 로그 6개월 이상 보관

채용 사이트 크롤링은 **각 사이트 약관을 위반하지 않는 범위**(사용자가 등록한 URL만 파싱)로 한정합니다.

## 셋업

```bash
pnpm install                       # 또는 npm install
cp .env.local.example .env.local   # 키 입력
supabase db push                   # 또는 SQL editor에 supabase/migrations/*.sql 붙여넣기
pnpm dev
```

## 폴더 구조
```
app/
  (auth)/        로그인·회원가입
  (app)/         인증 후 대시보드
  api/           서버 라우트 (파싱·발송·웹훅·옵트아웃)
lib/
  supabase/      Supabase 클라이언트(브라우저/서버/서비스)
  adapters/      채용 사이트별 HTML 파서
  email/         Resend 클라이언트 + 템플릿 렌더링
  validators/    Zod 스키마
supabase/
  migrations/    DB 스키마 + RLS 정책
```
